A project to implement POSIX.1e under Linux

The linux-privs@mit.edu mailing list was established in 1996 and some work has been done to implement POSIX.1e (formerly POSIX 6). Here is an old extract of the mailing list containing patches to old kernels: linux-privs@mit.edu.

The current state of the Linux-Privs specification document:

  • On-line documentation for Linux-Privs
  • A text version (includes highlighting which works when viewed with more but not Netscape).
  • PostScript [30 pages - some blank]
  • A tar.gz file of the html documentation
  • (sgml source)
  • All of the source code. Patches/libraries and basic apps.
