Welcome To The NQS Archive

This is the home of JISC project NTI/48.2 : Support for Distributed Batch Systems on UNIX. For more information about this work, see our World-Wide Web home page.

The following files are available here :

  • latest

    Contains symbolic links to the latest versions of the publically-available software.

    New users should download files using these links. Please read the README file in that directory for further information.

  • v3.36

    All releases of Monsanto-NQS v3.36.

  • v3.37

    All releases of Monsanto-NQS v3.37.

    PLEASE NOTE that there will be NO MORE releases under the name of Monsanto-NQS. All new releases will be under the new name of Generic NQS.

  • v3.40

    All releases of Generic NQS, v3.40.

  • v3:50

    All PUBLIC TEST releases of Generic NQS, v3.50. These releases do NOT compile, and are snapshots of on-going MAJOR surgery to the Generic NQS source tree, intended to produce a more robust, and portable version of NQS.
